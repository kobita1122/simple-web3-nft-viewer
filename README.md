# Simple Web3 NFT Viewer

This is a beginner-friendly NFT project.

This project helps you:
- Connect a crypto wallet
- Read wallet address
- Check blockchain network
- Fetch NFT token list
- Display NFT IDs on screen

This project uses:
- HTML
- JavaScript
- Public NFT API

## Features
- One-click wallet connect
- Show wallet address
- Show network name
- Fetch NFT token IDs
- Simple list display

## Requirements
- MetaMask installed
- Internet connection
- Basic JavaScript knowledge

## How to Run
1. Download this repository
2. Open `index.html` in browser
3. Click "Connect Wallet"
4. Click "Load NFTs"
5. See NFT token IDs on screen

## File Structure
- index.html -> Main webpage
- app.js -> App logic
- config.js -> API settings
- provider.js -> Web3 provider
- connect.js -> Wallet connect
- wallet.js -> Wallet reader
- network.js -> Network reader
- nft.js -> NFT fetch logic
- ui.js -> Screen updates
- utils.js -> Helper functions

## Learning Goal
This project helps beginners learn:
- How NFTs are stored on blockchain
- How to read wallet NFT data
- How Web3 websites work
